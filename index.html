The Challenge

Reinforcement learning underlies intelligent systems such as autonomous vehicles and game AI. Understanding its trial-and-error learning process is difficult without visualization. This simulator bridges that gap by showing behavior as it emerges.

Our Objective

Enable users to build custom grid maps, train AI agents, and see reinforcement learning principles applied in real time.

Learn how states, actions, rewards, and policies interact while watching the AI optimize its behavior.

Key Reinforcement Learning Concepts

  • State: The car's current position and map layout define the state.
  • Action: Moves the agent can take: up, down, left, or right.
  • Reward: Positive for reaching goals, negative for collisions or invalid moves.
  • Exploration vs Exploitation: Balancing trying new moves with using known successful actions.
  • Q-Table: A table mapping state-action pairs to expected cumulative rewards.
